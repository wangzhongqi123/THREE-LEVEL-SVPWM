# THREE-LEVEL-SVPWM

## 项目描述
### C代码
这是一个三电平空间矢量脉宽调制（SVPWM）等幅值算法实现，旨在优化电力电子控制系统的效率。该项目实现了三电平逆变器的 SVPWM 控制算法，适用于数字电源系统例如：T型,中点钳位型三电平逆变器、APF（有源电力滤波器)。
### 可视化工具
SVPWM_visualization_tool文件夹存放的是三电平svpwm可视化工具(完全匹配C语言算法,就相当于做了个前端界面),可以通过点击文件夹，在终端中运行然后依次执行#### npm install
#### npm run dev
把弹出来的本地地址复制到浏览器运行即可实现部署
这个可视化工具可以做教学,研究,算法分析,展示等等

## 功能
- 支持三电平逆变器的 **SVPWM 控制算法**。
- 实时输入电压调节。
- 支持硬件部署之前的所有逻辑（例如部署到 **TI C2000 EPWM** 或 **STM32G474 HRTIM** 等硬件，只差一个占空比逻辑，部署非常简单，欢迎联系我：wzqsie@outlook.com)
